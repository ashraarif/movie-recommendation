CineMate: Personalized Movie Recommendations¶

Presented by N.Ashra¶

iPEC Solutions Pvt.Ltd¶

In an era where digital streaming platforms have become the primary source of entertainment, the overwhelming abundance of content has made it increasingly difficult for users to find movies that match their individual tastes. CineMate is a personalized movie recommendation system designed to address this challenge by leveraging advanced machine learning algorithms and collaborative filtering techniques. The system analyzes user preferences, viewing history, and movie attributes to provide tailored movie suggestions that align with each user's unique interests.

def recommend(text):
    emb = embed([text])
    neighbors = nn.kneighbors(emb, return_distance=False)[0]
    return df["original_title"].iloc[neighbors].tolist()
